Bienvenido al Desarrollo Web

El desarrollo web es el proceso de crear sitios y aplicaciones que puedes ver en internet usando un navegador como Chrome o Firefox.

¿De qué están hechos los sitios web?

Todo sitio web se construye con dos ingredientes principales:

HTML — El Contenido

HTML es el lenguaje que define el contenido de la página: títulos, párrafos, imágenes y enlaces.

CSS — El Estilo

CSS es el lenguaje que da color, tamaño y forma a ese contenido. ¡Hace que todo se vea bonito!

Fórmula del Desarrollo Web

Sitio Web = Contenido (HTML) + Estilo (CSS)

Ejemplos de sitios web que ya conoces

YouTube, Wikipedia, Instagram y TikTok están todos construidos con HTML y CSS.
