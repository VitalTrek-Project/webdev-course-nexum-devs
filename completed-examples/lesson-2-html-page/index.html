¡Hola, Mundo!

Esta es mi primera página HTML. La creé aprendiendo desarrollo web desde cero.

¿Qué aprendí hoy?

Aprendí que una página HTML tiene tres partes principales: la etiqueta html, la sección head y la sección body.

Las etiquetas que usé

La etiqueta html envuelve todo el documento.

La etiqueta head contiene información de configuración como el título de la página.

La etiqueta body contiene todo el contenido visible que el usuario puede ver en el navegador.

Mi meta

Quiero aprender a crear sitios web increíbles usando HTML y CSS.
