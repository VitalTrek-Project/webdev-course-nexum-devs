# Lección 1: ¿Qué es el Desarrollo Web?

## ¿Qué es internet?

Internet es una red gigante de computadoras conectadas entre sí alrededor del mundo.

Cuando abres YouTube, Instagram o Wikipedia, estás accediendo a archivos guardados en computadoras remotas llamadas servidores.

## ¿Qué es un sitio web?

Un sitio web es un conjunto de páginas con información que puedes ver desde tu navegador.

Ejemplos de sitios web que ya conoces:

* YouTube
* Wikipedia
* Instagram
* TikTok

Todos ellos están construidos con los mismos ingredientes básicos: HTML y CSS.

## Los dos ingredientes de un sitio web

Todo sitio web en internet está hecho con dos lenguajes fundamentales.

### HTML — El Contenido

HTML significa *HyperText Markup Language* (Lenguaje de Marcado de Hipertexto).

Es el lenguaje que define el contenido de la página:

* Títulos
* Párrafos
* Imágenes
* Enlace

Piensa en HTML como el esqueleto de tu página web.

### CSS — El Estilo

CSS significa *Cascading Style Sheets* (Hojas de Estilo en Cascada).

Es el lenguaje que da apariencia visual al contenido:

* Colores
* Tamaños
* Fuentes
* Diseño y distribución

Piensa en CSS como la ropa y los colores que le pones al esqueleto.

## La fórmula del desarrollo web

```
Sitio Web = Contenido (HTML) + Estilo (CSS)
```

Sin HTML no hay contenido que mostrar.

Sin CSS la página funciona, pero se ve sin formato ni color.

Juntos crean la experiencia visual que ves cada vez que navegas por internet.

## ¿Qué es un navegador web?

Un navegador web es el programa que lee los archivos HTML y CSS y los convierte en la página visual que ves en pantalla.

Navegadores populares:

* Google Chrome
* Mozilla Firefox
* Safari
* Microsoft Edge

El navegador es tu herramienta principal en este curso.

## ¿Qué necesitas para empezar?

En este curso no necesitas instalar ningún programa.

Solo necesitas:

* Un navegador web
* Conexión a internet

Usaremos CodePen, una herramienta gratuita en línea que te permite escribir HTML y CSS y ver el resultado de inmediato.

## Ejemplo práctico

Abre CodePen y escribe el siguiente código en el panel HTML:

```html
<h1>Bienvenido al Desarrollo Web</h1>
<p>Todo sitio web tiene contenido (HTML) y estilo (CSS).</p>
```

Observa cómo el navegador muestra el título y el párrafo automáticamente.

## Resultado esperado

La página mostrará:

* Un título grande con el texto "Bienvenido al Desarrollo Web".
* Un párrafo debajo explicando la fórmula básica.

## Puntos clave de esta lección

* El desarrollo web es la creación de sitios y aplicaciones para internet.
* HTML define el contenido de una página web.
* CSS define la apariencia visual de esa página.
* La fórmula es: Sitio Web = Contenido (HTML) + Estilo (CSS).
* Solo necesitas un navegador para comenzar.

## Conclusión

Con solo dos lenguajes, HTML y CSS, puedes construir cualquier sitio web.

En las siguientes lecciones aprenderás paso a paso cómo escribir tu primer código HTML y darle estilo con CSS.
