# Lección 3: Formateando el texto

## ¿Qué son los encabezados (Heading tags)?

Los encabezados (Heading tags) permiten establecer títulos o subtítulos que se desean mostrar en la webpage. Los headings se establecen con las etiquetas `h1` hasta `h6`, siendo `h1` el más importante y `h6` el menos importante. Los buscadores utilizan los headings para indexar la estructura y contenido del webpage.

## Ejemplos de como usar los encabezados:

```html
<h1>Encabezado 1</h1>
<h2>Encabezado 2</h2>
<h3>Encabezado 3</h3>
<h4>Encabezado 4</h4>
<h5>Encabezado 5</h5>
<h6>Encabezado 6</h6>
```

## ¿Qué es un párrafo (Paragraph tag)?

Un paragraph (párrafo) siempre se establece con las etiquetas `p`, para poder iniciar en una nueva línea y es normalmente un bloque de texto. El browser se encarga automáticamente de agregar márgenes antes y después del párrafo.

## Ejemplo de como usar un párrafo:

```html
<p>Este es un párrafo de texto.</p>
```

## Ejemplo práctico:

```html
<h1>Copa Mundial de Fútbol de 2026</h1>
<h2>Selección de Argentina</h2>
<p>Tri campeón</p>
```

## Resultado esperado:

La página mostrará:
* Este encabezado principal: Copa Mundial de Fútbol de 2026
* Este subencabezado: Selección de Argentina
* Este párrafo: Tri campeón

## Conclusión:

Los heading tags permiten establecer títulos o subtítulos que se desean mostrar en un página web, mientras que los párrafos sirven para organizar y mostrar bloques de texto dentro de la página.
